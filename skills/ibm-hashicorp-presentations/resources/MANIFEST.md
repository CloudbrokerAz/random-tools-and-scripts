## Notes

- The `.fntdata` files are OOXML-obfuscated font files. Each file's first 32 bytes are XOR'd
  with a GUID-derived key. To use IBM Plex Sans in presentations, install the font from
  the official IBM Plex repository instead of trying to deobfuscate these files.
- EMF files are Windows Enhanced Metafile vector graphics. They render natively in PowerPoint
  but may need conversion for use in other tools.
- SVG icons follow the IBM Carbon Design System icon grid (32x32 viewbox for small icons,
  ~80-90px viewbox for large pictograms).
- Images sourced from the POTX are primarily 1920x1080 cover backgrounds.
  Images from the PPTX include a wider variety of sizes for different layout types.
- Two files (image1.emf and image2.png) were identical in both templates and were deduplicated.
